## Usage

- instalation

```
npm install sancode86/favemoji
```

Once installed, it's simple and easy to use:
- Pass variable of the emoji as string:
(See available emojis!)

```
<script>
  favemoji("//EMOJI");
</script>
```

- Get a random emoji:

```
<script>
  favemoji();
</script>
```

- Title (optional):
You can also specify document's title:

```
<script>
  favemoji("smile", "Hello!");
</script>
```
